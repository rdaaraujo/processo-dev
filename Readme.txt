[PT-BR]

Serviço de gestão de Clientes básico, criado em VB6 (PSF Clientes).

1- Menu didático com as opções 'Novo', 'Editar', 'Apagar' e 'Fechar' funcionais.

2- Importação de clientes via Lote (em massa), selecionando o arquivo desejado (.csv, .txt, .xlsx e etc..) na máquina,
sendo possível a importação ou cancelamento desta!

3- Delete e Alteração de cliente no BD, através da ID deste cliente onde a situação deste de 'ABERTO' para 'BAIXADO'.

4- Delete de todos os clientes no BD, através de 1 clique (podendo ser cancelado na caixa de diálogo).

5- View dos clientes presentes no BD (somente visualização).

6- API foi criada e executa 'perfeitamente' todo o processo do crud solicitado, apenas não foi realizada a conexão desta ao programa em VB6.

[EN-US]

Basic Customer Management Service, developed in VB6 (PSF Cliente).

1- User-friendly menu with functional options: 'New', 'Edit', 'Delete', and 'Close'.

2- Mass client import feature, enabling users to select desired file (.csv, .txt, .xlsx, etc..) on the machine, allowing either importation or cancellation.

3- Ability to delete and modify client records in the database, using the client's ID to change the status from 'ABERTO' to 'BAIXADO'.

4- Delete all clients in the database with a single click (with option to cancel in the dialog box).

5- View of clients present in the database (for visualization purposes only).

6- The API was created and performs 'perfectly' the entire CRUD process requested, only the connection to the VB6 program has not been implemented.
